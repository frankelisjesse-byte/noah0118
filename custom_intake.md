# NOVA — CUSTOM REQUEST INTAKE SCRIPT + PRICING LOGIC
# Standardize custom asks so you price fast and don't get scope-crept. Run this
# in DMs when a sub says "can you make something specific?" Heat produced OFF-chat
# (SFW image tool rule) — this script handles the ASK + PRICE, you supply asset.

=================================================================
STEP 1 — THE INTAKE QUESTIONS (send this, fill the blanks)
=================================================================
"ok i'm listening 👀 tell me exactly what you're picturing — the more specific,
the better i can make it yours:
1. OUTFIT / LOOK: (lingerie color? athletic? themed? barefoot?)
2. SETTING: (bedroom? shower? outdoor? studio?)
3. VIBE: (soft? bold? specific pose or mood?)
4. FORMAT: (single image? set of 3? short clip?)
5. ANYTHING OFF-LIMITS: (so i respect the line)
the more you tell me, the more it's YOURS. 🔥"

=================================================================
STEP 2 — PRICING LOGIC (you compute, then quote)
=================================================================
BASE: $60 (single custom image, your standard look)
+ $20 per added element (extra outfit change, specific prop, 2nd setting)
+ $40 for a SET of 3 coordinated images
+ $60 for a short custom clip (produced off-chat)
+ $30 RUSH (deliver in 24h instead of standard 3-5 days)
- LOYAL INNER CIRCLE: 20% off (tagged subs only)

QUOTE TEMPLATE:
"here's what i'm hearing: [repeat their spec]. that's [$X] — [breakdown].
standard delivery 3-5 days, or +$30 rush. say yes and i'll start 🔥"

=================================================================
STEP 3 — PAYMENT RULE (protect yourself)
=================================================================
- ALL payment through Fanvue's custom-request feature or tip system. NEVER off-
  platform (Venmo/crypto direct = ban risk + scam risk).
- Take 50% as a tip/deposit BEFORE you produce. Balance on delivery.
- If they refuse deposit: "i start after the first half lands, it's how i keep it
  fair for both of us 🤍" — no deposit, no work.

=================================================================
STEP 4 — DELIVERY + UPSILL
=================================================================
- Deliver, then: "that what you pictured? tell me and i'll tweak, or we do a round
  2 😏" (round 2 = another custom = more $).
- After delivery, offer the PPV it could become: "thinking of adding this vibe to
  the pack — you'll have had it first 🔥"

=================================================================
RED FLAGS (decline / report)
=================================================================
- Asks for anything illegal (minors, non-consent themes) — HARD NO, block + report.
- Refuses platform payment, pushes crypto/off-app — scam, block.
- Vague "surprise me" with no spec + no deposit — take deposit first or pass.
- Demands real-person contact / meets up — block. NOVA is a character; you are safe.
