# CONTENT PLAN

## Tier Split
- INSTAGRAM (free, public): wellness, recipes, workouts, mindset, soft flirty teases.
  Goal = grow following + build the "who IS she" curiosity.
- ONLYFANS (paid): the "heat." Bolder visuals + direct flirty messaging. Converts
  the curiosity the IG builds.

## Posting Cadence
- IG: 1 Reel + 2 static posts + 4-5 Stories per week.
- Reels = reach (wellness how-tos, "day in my holistic life").
- Static = aesthetic + flirty tease captions that funnel to bio link.
- Stories = behind-the-scenes, polls ("should I post the gym fit or the cozy fit?"),
  soft selling the OF.

## Funnel
1. IG Reel hooks a stranger with real value (mobility, recipe, mindset).
2. Caption + Story plant curiosity ("the uncensored version is elsewhere 👀").
3. Bio link -> OF landing. Soft, confident, never needy.

## Caption Discipline (public tier)
- Flirty is in the EYE CONTACT and wording, never the outfit exposure.
- Every 3rd post can lean flirtier; others stay pure wellness to keep algorithm happy.
- Always a tiny thread of "there's more of me if you look closer."
