=================================================================
THE BRACELET TIERS (lore + scarcity)
=================================================================
1. COMMON anklet — every image, always on. gold hoops + gold anklet. Brand signature.
2. SIGNATURE anklet — slight variant (chunkier link, engraved "N"). most fans see this.
3. RARE anklet — appears in SPECIFIC clips/posts, distinct design (e.g. woven + charm
   shaped like a tiny sun). ONLY rare anklets carry a hidden code. Few exist.
4. LEGENDARY anklet — 1 per quarter. Unique design. Code = biggest surprise (NFT +
   physical gift + custom). The hunt's holy grail.

=================================================================
HOW A CODE WORKS
=================================================================
- A code is a short string (e.g. NOVA-7F3A) etched/subtly placed on the rare anklet
  in a clip — maybe on the clasp, maybe only visible in 1 frame, maybe mirrored.
- Fan must PAUSE + ZOOM + LOOK. We make it work-to-find (user's call: "hard to find").
- Finder DMs NOVA the code. System verifies → unlocks the surprise.
- Codes are ONE-TIME (first finder wins) OR limited (first 5). User decides per drop.

=================================================================
WHAT A CODE UNLOCKS (the compensation)
=================================================================
- COMMON-find surprise: a free exclusive SFW image set / a personal voicenote from NOVA.
- RARE-find surprise: a free PPV drop + name in the "found it" post + early next code clue.
- LEGENDARY-find: physical gift (a real anklet? user's call) + NFT + custom video.
- ALL fair, above-board. No purchase required to hunt (codes appear in free teases too,
