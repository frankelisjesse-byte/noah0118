=================================================================
A. THE VARIABLES WE TEST (the "what they prefer" list)
=================================================================
Each variable = an A/B (or A/B/C) split. We rotate, not guess.

1. VOICE / TONE (for any video VO or caption "voice")
   - V1 Whisper-soft / intimate ("just between us")
   - V2 Bright / bubbly / high-energy
   - V3 Sassy / confident / playful tease
   - V4 Deadpan / dry-humor (our terrible-puns mode)
   Measure: which tone gets more completion + comments + DMs.

2. SMILE / FACIAL ENERGY
   - S1 Soft gentle smile (warmth)
   - S2 Beaming open smile (joy)
   - S3 Smirk / half-smile (mystery/flirt)
   - S4 Neutral / contemplative (moody)
   Measure: which drives profile visits + saves.

3. OUTFIT / LOOK
   - O1 Athleisure / gym (wellness anchor)
   - O2 Casual home (relatable — sock-drawer Sundays)
   - O3 Dressed-up / night-out (aspiration)
   - O4 Feet-focused (our differentiator — barefoot / anklet hero)
   Measure: which drives link clicks + Fanvue visits.

4. MOTION FORMAT
   - M1 Dance / movement (pre-workout dance habit)
   - M2 Static pose / beauty still
   - M3 Talking-head / direct-to-camera
   - M4 POV / scene (caught-in-the-wild Phase-0 style)
   Measure: completion rate + rewatches (loop value).

5. CAPTION STYLE
   - C1 Question-close only (our locked algo engine)
   - C2 Story snippet (habit reveal)
   - C3 Hot-take / controversial-but-safe
   - C4 Ultra-short / mysterious
   Measure: comments per post.

6. HOOK TYPE (first 1–2 sec)
   - H1 Pattern interrupt ("stop doing X")
   - H2 Curiosity gap ("you won't believe what...")
   - H3 Relatable ("POV: it's sock-drawer Sunday")
   - H4 Direct address ("hey, you — yes you")
   Measure: hold-through-3-sec rate (proxy = completion).

=================================================================
B. INSTRUMENTATION (how we capture the data)
=================================================================
We already have kpi_scorecard_30day.md. Extend it with a PER-POST TAG sheet:
  Columns: date | platform | variable values (V/S/O/M/C/H codes) | reach |
  profile_visits | link_clicks | fanvue_visits | new_subs | saves | shares |
  comments | completion_est (for video)
Lightweight: one row per post. Do it in a Google Sheet or the phone Notes app
+ weekly dump. The goal is a clean dataset by Day 90.

DAILY (5 min): log the post row.
WEEKLY (30 min): aggregate by variable code, find the leader per metric.
MONTHLY: write a "what won" memo → feeds Academy curriculum.

=================================================================
C. TEST DESIGN (so the result is real, not noise)
=================================================================
- Minimum N: ~10 posts per variant before calling a winner (statistical smell,
  not rigor — we're lean, not a lab).
- Hold everything ELSE constant when testing one variable (only change voice,
  keep outfit/smile/motion same) so we isolate the effect.
- Rotate variants on a schedule (e.g., voice test weeks 1–3, smile test 4–6).
- Watch for CONFOUNDS: a viral sound can mask a bad smile. Note externalities.
- Decision rule: variant with highest (link_clicks × new_subs) per 1k reach
  WINS the conversion test; highest (saves + shares) wins the reach test.

=================================================================
D. WHAT "WINNING" LOOKS LIKE PER METRIC
=================================================================
- Reach: trending up week over week (algorithm learning our audience).
- Completion: >50% on Reels/TikTok = healthy; >70% = hero.
- Saves: the silent winner — high saves = evergreen reach.
- Comments: our question-close should pull 2–5% of reach as comments early.
- Link clicks: 1–3% of profile visits is normal; >5% = hot creative.
- New subs: the only metric that pays. Optimize the STEP before it first.
