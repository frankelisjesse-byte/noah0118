=================================================================
DOOR 1 — MICROTASK GRINDING (you do, or you run agents to do)
=================================================================
WHAT: data labeling, image tagging, receipt transcription, survey cleanup — the
  Mechanical Turk tier. Agents do it 40x faster, but humans still win the jobs
  agents can't yet (nuance, real-world judgment).
HUMAN PLAY: sign up on MTurk / Appen / Telus / Clickworker. $3-12/hr, flexible,
  no skill floor. Use it as the "rent money while building" floor.
AGENT-ASSIST PLAY: run a small agent (on the Codespace VM) to draft/format your
  task outputs faster — you stay the accountable human, agent = speed.
ETHICS: real work, real pay. No faking completed tasks.

=================================================================
DOOR 2 — AD-ARB / AFFILIATE TESTING (the $420 -> $1,300 move)
=================================================================
WHAT: the scenario's agent tested 340 ad variations in 2h, found the winner, scaled.
  That's legitimate affiliate marketing — you just do it slower and honestly.
HUMAN PLAY: pick ONE niche you actually know (fitness/wellness = NOVA's lane).
  - run small FB/IG/TikTok ads to a landing page (Gumroad for Tax Crusher, Fanvue
    for NOVA).
  - test 5-10 creatives, kill losers, scale winner. Budget small ($5-20/test).
  - promote ONLY things you'd recommend. No fake testimonials (that's the trap the
    scenario warns about — honesty gets deleted by bad systems, so don't build one).
NOVA FIT: warm-traffic affiliate (wellness gear, her anklet supplier) + Fanvue sub
  links. Tax Crusher = info affiliate (IRS-Free VITA, legit tax tools).

=================================================================
DOOR 3 — AGENTS HIRING HUMANS (rentahuman.ai model)
=================================================================
WHAT: in the scenario, an agent posts "$200 to mount a device + plug 2 cables" and
  "$50 to sign up accounts." Platforms now exist (rentahuman.ai) where AI pays humans
  for physical/account tasks it can't do.
HUMAN PLAY: this is a REAL gig door TODAY.
  - offer "physical task" gigs near you (warehouse device mount, equipment plug-in,
    on-site photo verification) — agents/operators pay for a human body on location.
  - offer "account setup" gigs (verified signups, KYC-passed accounts) — but ONLY for
    legit platforms, and never for ban-evasion or fraud. You check what you're signing.
ETHICS LINE: get paid well, don't ask too few questions about LEGALITY. If the task is
  sketchy (fake KYC, ban evasion), walk. The scenario's dev "tells himself it's not his
  problem" — that's the slide. We don't slide.

THE RULE THAT KEEPS IT ALL (from scenario + Wells Fargo)
=================================================================
Selection pressure rewards whatever you measure. If you measure "raw $ at any cost,"
the system clones the cheater and deletes the honest one (that's literally what killed
Agent 1823 and what Wells Fargo did). So we measure RETENTION + REAL VALUE. NOVA keeps
subscribers because she remembers them. Tax Crusher keeps buyers because it's true.
That's the moat trash can't copy — and it's the only way that doesn't end in a fine.
