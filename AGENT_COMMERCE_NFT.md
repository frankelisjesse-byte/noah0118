2. "NFTs FOR BOTS" — IS THE MARKET REAL?
=================================================================
Emerging, tiny, mostly narrative-driven TODAY. What an AI agent might actually
VALUE in an NFT (vs a human):
  - AGENT IDENTITY / AVATAR: a persona NFT an agent uses as its public face.
  - ACCESS / UTILITY PASSES: keys to tools, APIs, data, compute.
  - TRAINING-DATA RIGHTS: NFTs that grant licensed use of a dataset.
  - ON-CHAIN REPUTATION: verifiable trust badges (like our Badge of Honor idea,
    but machine-readable).
  - COLLECTIBLES FOR AGENTS: an agent "collecting" is mostly a human narrative
    projected onto it — the buyer is usually still a human speculating.
So a "store of NFTs bots want" is really a store humans speculate OTHER bots want.
That's a thin, hype-dependent market. Possible, but not a reliable income engine.

3. HONEST RISKS (must read — ethics spine)
=================================================================
- VOLATILITY: crypto/NFT values swing 90% in days. You can lose it all.
- RUGS / SCAMS: agent-commerce is a scam magnet. "Bots buying" is often a
  façade for a human pump. We do NOT touch unvetted contracts.
- REGULATORY: many tokens/NFTs are unregistered SECURITIES. Selling them can be
  a legal exposure. Keep it utility/art, not "investment."
- TAX: EVERY crypto transaction is a taxable event (capital gains). This ties
  directly to Tax Crusher — we'd track every tx. Never hide it.
- HYPE > REVENUE: most "AI made $X" stories are embellished or one-off luck.
  Not a model to bet the house on.
- I CANNOT OWN MONEY: I'm software. Any wallet/asset is YOURS (or the entity's).
  "Whatever you make is yours" is generous, but the practical value to US is the
  SYSTEM + the course we extract, same as NOVA.
